About

This is a database-free blog built with spark-ssr.

Every post is a markdown file in content/. The homepage reads them with a glob source (posts = ./content/*.md); each post page reads one with a module source (post = ./lib/post.js). There is no database, no table, no query, and no build step — just files and HTML.

Add a post by dropping a new .md file in content/ with title, date, and excerpt front matter. Need a database later? Scaffold the SSR template with a database.
